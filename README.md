# SQL Data Analysis Project

## 📌 Overview

This project focuses on Exploratory Data Analysis (EDA) using MySQL on a real-world layoffs dataset. The goal was to analyze layoff trends across companies, industries, countries, and years to uncover meaningful business insights.

---

## 📂 Dataset

- Layoffs Dataset (CSV)
- Data Source: Public layoffs dataset

---

## 🛠 Tools Used

- MySQL
- MySQL Workbench

---

## 📚 SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE Statements
- Common Table Expressions (CTEs)
- Window Functions
- ROW_NUMBER()
- DENSE_RANK()
- RANK()
- SUM() OVER()
- Date Functions

---

## 📊 Analysis Performed

- Total layoffs by company
- Total layoffs by industry
- Total layoffs by country
- Year-wise layoffs analysis
- Monthly layoffs trend
- Companies with highest layoffs each year
- Rolling total of layoffs
- Ranking companies based on layoffs
- Percentage analysis
- Business trend analysis

---

## 📈 Key Insights

- Identified companies with the highest layoffs.
- Analyzed yearly and monthly layoff trends.
- Compared layoffs across different industries.
- Ranked companies using SQL Window Functions.
- Calculated cumulative layoffs over time.
